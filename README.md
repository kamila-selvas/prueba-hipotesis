# App de Prueba de Hipótesis

Aplicación interactiva desarrollada con Streamlit para lamateria de Probabilidad y Estadística.

## Funcionalidades
- Carga de datos CSV o generación sintética
- Visualización con histograma y boxplot
- Prueba de hipótesis Z con gráfica de región crítica
- Asistente de IA con Google Gemini

## Tecnologías
- Python
- Streamlit
- NumPy / Pandas / SciPy / Matplotlib
- Google Generative AI (Gemini)

## Cómo ejecutar
```
streamlit run app.py
```
